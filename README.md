## 📖 프로젝트 소개

스마트 냉장고 비용이 부담스러운 1인가구나 가정을 타겟<br>
유통기한 알림과 재료관리, 레시피 추천까지 가능한 어플케이션<br>
인공지능을 추가하여 보다 뛰어나고 쉬운 사용 방향성 제시

## ✔ 주요기능
+ 회원관리
  + 회원가입
  + 로그인
  + 로그아웃

+ 식재료 관리
  + 식재료 추가
    + 카메라를 통한 자동 인식 추가
    + 식재료 리스트에서 검색하여 추가
   + 식재료 삭제
   + 식재료 유통기한 확인
   + 식재료별 알러지 관리
   + 유통기한 임박 알람

+ 레시피 조회
  + 식재료 선택
  + 식재료 검색
  + 상세 레시피 슬라이드 제공
  + 자동 타이머 설정

+ 알러지 관리
  + 사용자별 알러지 추가/삭제

+ 냉장고 코드 관리
  + 냉장고 코드 복사, 변경, 초기화
  + 냉장고 코드 암호화, 복호화
